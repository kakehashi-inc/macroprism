MacroPrism is an Electron + React (MUI) desktop GUI for starting/stopping, monitoring, logging, and exposing arbitrary local processes via ngrok. Its core is generic process management; the optional `mcp-auth-proxy` integration is an auxiliary feature attachable to a managed process (which also covers use cases like running MCP servers). Managed entities are called "processes" everywhere (UI labels, types like `ProcessConfig`/`Processes`, and the `processes` key in config.json; legacy `mcpServers`/`servers` keys are auto-migrated on load). The repo is organized into `src/main` (Electron main process, IPC handlers, and services like Config/Process/Log/Ngrok/HttpsProxy managers), `src/preload` (safe IPC bridge), and `src/renderer` (React + Zustand UI pages/components). App data lives under `~/.mcpm` (config, certs, logs) — `mcpm` is an internal identifier from the consonants of MacroPrism.
